## Dependencias

## Configuraciones de Vscode

En caso de usar vscode recomiendo copiar los settings de `.vscode/settings.example.json` a tu workspace.

```bash
cp .vscode/settings.example.json .vscode/settings.json
```

## Local Development

```bash
pnpm install
```
