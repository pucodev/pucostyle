# Input

## Uso b√°sico

Para implementar inputs usar la clase `input` y utilizar `field` como clase contenedora.

<PreviewCode>
  <div className="is-flex is-flex-column gap-3">
    <div className="field">
      <label className="label">Name</label>
      <input className="input" type="text" placeholder="Text input" />
    </div>
    <div className="field">
      <label className="label">Name</label>
      <input className="input" type="text" placeholder="Text input" />
      <p className="help">Help message</p>
    </div>
  </div>
</PreviewCode>

```html
<div class="is-flex is-flex-column gap-3">
  <!-- Normal input -->
  <div class="field">
    <label class="label">Name</label>
    <input class="input" type="text" placeholder="Text input" />
  </div>

  <!-- With help message -->
  <div class="field">
    <label class="label">Name</label>
    <input class="input" type="text" placeholder="Text input" />
    <p class="help">Help message</p>
  </div>
</div>
```
