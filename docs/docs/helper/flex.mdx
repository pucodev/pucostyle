# Flex

Clases para manejar flex items

## Uso básico

Para poder usar flex items usar la clase `is-flex`

<PreviewCode>
  <div className="is-flex text-white gap-3">
    <div className="bg-primary p-3 rounded-md">Flex item 1</div>
    <div className="bg-accent p-3 rounded-md">Flex item 2</div>
    <div className="bg-primary p-3 rounded-md">Flex item 3</div>
    <div className="bg-accent p-3 rounded-md">...</div>
  </div>
</PreviewCode>

```html
<div class="is-flex gap-3">
  <div>Flex item 1</div>
  <div>Flex item 2</div>
  <div>Flex item 3</div>
  <div>...</div>
</div>
```

## Flex direction

### Row

Usa `is-flex-row` para configurar la dirección horizontal

<PreviewCode>
  <div className="is-flex is-flex-row text-white gap-3">
    <div className="bg-primary p-3 rounded-md">Flex item 1</div>
    <div className="bg-accent p-3 rounded-md">Flex item 2</div>
    <div className="bg-primary p-3 rounded-md">Flex item 3</div>
    <div className="bg-accent p-3 rounded-md">...</div>
  </div>
</PreviewCode>

:::tip

Por defecto usando solo `is-flex` ya está configurado como flex row

:::

### Column

Usa `is-flex-column` para configurar la dirección vertical

<PreviewCode>
  <div className="is-flex is-flex-column text-white gap-3">
    <div className="bg-primary p-3 rounded-md">Flex item 1</div>
    <div className="bg-accent p-3 rounded-md">Flex item 2</div>
    <div className="bg-primary p-3 rounded-md">Flex item 3</div>
    <div className="bg-accent p-3 rounded-md">...</div>
  </div>
</PreviewCode>

```html
<div class="is-flex is-flex-column gap-3">
  <div>Flex item 1</div>
  <div>Flex item 2</div>
  <div>Flex item 3</div>
  <div>...</div>
</div>
```

## Justify content

| class                              | property                         |
| ---------------------------------- | -------------------------------- |
| `is-justify-content-flex-start`    | `justify-content: flex-start`    |
| `is-justify-content-flex-end`      | `justify-content: flex-end`      |
| `is-justify-content-center`        | `justify-content: center`        |
| `is-justify-content-space-between` | `justify-content: space-between` |
| `is-justify-content-space-around`  | `justify-content: space-around`  |
| `is-justify-content-space-evenly`  | `justify-content: space-evenly`  |
