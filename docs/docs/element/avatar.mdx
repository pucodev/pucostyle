# Avatar / Image

Agrega estilos a tus imágenes

## Uso básico

<PreviewCode>
<div className='is-flex gap-4'>
  <figure className="image is-size-128">
    <img
      src="https://source.unsplash.com/random/200x200/?profile"
      alt="Imagen de perfil"
    />
  </figure>

<figure className="image is-size-128 is-rounded">
  <img
    src="https://source.unsplash.com/random/200x200/?profile"
    alt="Imagen de perfil"
  />
</figure>

<figure className="image is-size-128 is-rounded-3">
  <img
    src="https://source.unsplash.com/random/200x200/?profile"
    alt="Imagen de perfil"
  />
</figure>

  <figure className="image is-size-128 is-rounded-3">
    <div className="fallback">PS</div>
  </figure>
</div>
</PreviewCode>

## Redondeado

Para redondear las imágenes puedes usar el helper de dimensiones para ello usa las clases `is-rounded-*`

Donde `*` puede ser del 1 al 12, para más información visita la [documentacion de personalizar espacios](../customize/spacing)

En caso se requiera que este completamente redondeado puedes usar la clase `is-rounded`

<PreviewCode>
<div className='is-flex gap-4'>

## Fallback

Muestra un texto debajo de tu imágen mientras se descarga, por ejemplo lo puedes usar para colocar las iniciales del usuario

<PreviewCode>
  <div className="is-flex gap-4">
    <figure className="image is-size-128 is-rounded-3">
      <div className="fallback">PS</div>
      <img src="" alt="" />
    </figure>
  </div>
</PreviewCode>

```html
<figure class="image is-size-128 is-rounded-3">
```
