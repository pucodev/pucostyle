## Divider vertical

Para colocar el divider en vertical usar la clase `is-vertical`

<PreviewCode>
  <div className="is-flex">
    <div className="divider is-vertical px-3 py-2">

    </div>
    <div className="divider is-vertical px-3 py-2">
      <div>Hola</div>
    </div>
    <div className="px-3 py-2">
      <div>Hola</div>
    </div>
  </div>
</PreviewCode>
