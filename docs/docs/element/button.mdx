# Button

## Uso básico

Para usar botones usar la clase `btn`

<PreviewCode>
  <div className="is-flex gap-3">
    <button className="btn">Button 1</button>
    <a className="btn" href="#">
      Button 2
    </a>
  </div>
</PreviewCode>

```html
<div class="is-flex gap-3">
  <button class="btn">Button 1</button>
  <a class="btn" href="#">Button 2</a>
</div>
```

## Colores

Un botón está disponible para varios botones para ello se debe usar la clase `is-{color}` junto con `btn`

export const ColorRow = ({ color }) => (
  <tr>
    <td>
      <CodeBlock>is-{color}</CodeBlock>
    </td>
    <td>
      <PreviewCode>
        <button className={`btn is-${color}`}>Button</button>
      </PreviewCode>
    </td>
  </tr>
)

<table>
  <thead>
    <tr>
      <th>Clase</th>
      <th>Preview</th>
    </tr>
  </thead>
  <tbody>
    {colors.map((color, idx) => (
      <ColorRow color={color} />
    ))}
  </tbody>
</table>

## Outlined

Un botón puede ser del tipo **outlined** usando la clase `is-outlined` además se puede seleccionar el color usando la clase `is-{color}`
