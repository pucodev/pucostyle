# Dialog

Un **dialog** está constituido por:

- `dialog`: contenedor
  - `dialog-bg`: Overlay con opacidad que ocultará el contenido del fondo
  - `dialog-card`: contenedor del dialog
    - `dialog-header`: Cabecera del dialog, donde va el título y el botón de cerrar
    - `dialog-body`: Donde se agrega el conenido del dialog
    - `dialog-footer`: Donse se agrega los botones de acción

## Uso básico

:::info

Recuerda que para activar el dialog se debe agregar la clase `is-active`

:::

<PreviewCode>
